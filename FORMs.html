📅 Date Picker type="date"
Select or type a date

Stored value: YYYY-MM-DD  ·  e.g. 2024-07-15

—
<input type="date" id="d" />

// Read in JS:
var v = document.getElementById("d").value;
// v = "2024-07-15"
new Date(v + "T00:00:00");
🕐 Time Picker type="time"
Select or type a time

—

// Value is always 24h format:
// "08:30" or "14:45"

📆 Date & Time Combined datetime-local
Pick date and time together
